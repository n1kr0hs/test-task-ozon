# Тестовое задание "Реализовать блок Progress"

##  Реализованный функционал

- **Плавное заполнение** прогресса при изменении значения (анимация через CSS transition)
- **Анимация вращения** круга (включается/выключается чекбоксом)
- **Состояние ошибки**: при значении ≥ 101 круг полностью заполняется красным цветом
- **Скрытие/показ** компонента (чекбокс Hide)
- **Переиспользуемость**:
  - Не привязан к конкретным ID
  - Не меняет классы или стили внешнего контейнера
  - Можно создавать несколько независимых экземпляров с разными параметрами (размер, цвета, толщина линии и т.д.)
- Полностью изолированный компонент (использует callback вместо глобальных событий)
- Поддержка corner cases: отрицательные значения → 0, NaN → 0, значения > 100 → ошибка
