# Тестовое задание "Реализовать блок Progress"

##  Реализованный функционал

- Плавное заполнение прогресса при изменении значения (через input)
- Анимация вращения круга (включается чекбоксом «Animate»)
- Состояние ошибки: при value ≥ 101 круг полностью заполняется и становится красным
- Скрытие/показ компонента (чекбокс «Hide»)
- Переиспользуемость: компонент не привязан к конкретному id, не меняет классы внешнего контейнера, можно создавать несколько независимых экземпляров с разными размерами и цветами
